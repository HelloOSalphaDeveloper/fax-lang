​Fax Programming Language 📠
​Добро пожаловать в репозиторий Fax — компактного стекового языка программирования, созданного с нуля!
​🏗️ Архитектура
​Проект состоит из двух главных частей:
​Compiler (compiler.py): переводит ваш код .fax в компактный бинарный байт-код .faxc.
​Virtual Machine (vm.c): исполняемая среда, которая читает байт-код и выполняет операции на стеке.
​🚀 Быстрый старт
​1. Сборка виртуальной машины:
gcc vm.c -o faxvm
2. Компиляция вашей программы:
python compiler.py
3. Запуск:
faxvm program.faxc
